# DENIS FBI STORE - Gestion Commerciale & Stock

Ce projet est une application web de gestion commerciale et de stock.

## Structure du Projet

- **`frontend/`** : Contient l'interface utilisateur, les vues et les fichiers sources (CSS, JS).
- **`backend/`** : Contient la logique serveur, les configurations et les actions.
- **`database/`** : Contient le schéma de la base de données et les scripts de migration.

## Installation

1. Clonez le dépôt.
2. Importez `database/database.sql` dans votre serveur MySQL.
3. Configurez `backend/config/db.php` avec vos identifiants.
4. Accédez à `frontend/index.php` via votre serveur web.
